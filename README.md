## Getting Started

### Dependencies
* firebase 8.10.0
* express
* ejs
* stripe

### Installing

Create a **.env** file and save the following variables:
* STRIPE_PUBLIC_KEY
* STRIPE_SECRET_KEY
* FIREBASE_API_KEY

### Executing program

1. Run the following command in terminal:
```
node server.js
```
2. Open https://localhost:3000 in your browser
